# House of Tartarus Asset Library

This repository is the authoritative source for House of Tartarus branding, card artwork, interface components, decorative elements, and editable source files.

## Production asset groups

- `assets/ui/` - interface components such as buttons, windows, panels, icons, cursors, inventory frames, scrollbars, and progress bars.
- `assets/heraldry/` - shields, crests, tridents, monograms, and rank emblems.
- `assets/banners/` - pennants, ribbons, flags, and long-form banners.
- `assets/badges/` - member, role, status, achievement, and rank badges.
- `assets/seals/` - wax seals, stamps, and embossed marks.
- `assets/textures/` - reusable leather, parchment, stone, metal, and pattern textures.
- `assets/mockups/` - assembled presentation examples and device or print previews.
- `assets/source/` - editable source documents used to create production exports.

## Working rules

1. Store finished reusable assets in the appropriate production directory.
2. Store editable XCF, SVG, ODG, or other source documents under `assets/source/` or the relevant project source directory.
3. Use lowercase snake_case filenames for production assets.
4. Use PNG for raster assets that require transparency.
5. Do not add proof sheets to production folders.
6. Keep temporary exports, scratch files, and experiments out of Git.

See `docs/NAMING-STANDARD.md` and `docs/ASSET-PRODUCTION-STANDARD.md` for required naming and export rules.
