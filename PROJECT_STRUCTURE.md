# Project Structure

## Branding

- `assets/branding/logos/` - primary House of Tartarus logos.
- `assets/branding/watermarks/` - transparent and low-opacity marks.
- `assets/branding/qr-codes/` - production QR codes and source data.

## Business cards

- `assets/cards/source/` - original composite artwork and editable source files.
- `assets/cards/front/` - cropped front-card artwork.
- `assets/cards/back/` - cropped back-card artwork.
- `assets/cards/exports/png/` - raster exports.
- `assets/cards/exports/pdf/` - printable PDF exports.
- `templates/avery-28878/` - Avery 28878 templates and populated master files.
- `members/` - member-specific card configurations and exports.

## Documentation and automation

- `docs/` - design notes, printing instructions, and standards.
- `scripts/` - build, validation, and export scripts.

## Future website

The website platform has not been selected.

- `website/shared/assets/` - reusable logos, images, copy, and other assets.
- `website/static-html/` - reserved for a static HTML/CSS/JavaScript site.
- `website/wordpress/` - reserved for a WordPress theme, plugin, or deployment files.

Only one website implementation needs to be developed. The unused directory can remain as a placeholder or be removed after the platform decision.
