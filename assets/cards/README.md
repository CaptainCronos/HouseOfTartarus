# Card Artwork

Recommended workflow:

1. Preserve the original composite artwork in `source/`.
2. Crop and export the card front into `front/`.
3. Crop and export the card back into `back/`.
4. Place the finished artwork into the Avery 28878 master template.
5. Save printable proofs in `exports/pdf/`.
6. Save high-resolution image exports in `exports/png/`.
